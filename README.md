# PlacesFlow_MERN
A full-stack MERN application for users to create, share, and manage their favorite places with integrated geocoding capabilities
PlacesFlow MERN
A robust, full-stack MERN (MongoDB, Express, React, Node.js) application designed to allow users to create, share, and manage their favorite places with integrated geocoding capabilities.

🚀 Overview
PlacesFlow is a complete solution for social location management. It empowers users to document their travels, pin locations on an interactive map, and interact with a community-driven database of interesting places.

🛠 Tech Stack
Frontend: React.js, CSS3

Backend: Node.js, Express.js

Database: MongoDB (via Mongoose)

Features:

User Authentication & Authorization

Interactive Geocoding (Nominatim integration)

Dynamic Place Management (Create, Update, Delete)

Responsive UI for seamless browsing

⚙️ Installation & Usage
Clone the repository:

Bash
git clone https://github.com/lordx2008x-ship-it/PlacesFlow_MERN.git
Setup Backend:

Navigate to back-end/

Run npm install

Configure your .env file (Database URI, JWT Secret)

Start server: npm start

Setup Frontend:

Navigate to front-end/

Run npm install

Start application: npm start
